Manufacturing Execution System

Como nivel superior de la arquitectura de automatización se desarrolló un Manufacturing Execution System (MES), encargado de consolidar la información proveniente del proceso productivo y transformarla en indicadores de desempeño para apoyar la toma de decisiones. El sistema permite visualizar en tiempo real el estado de la producción y calcular automáticamente el indicador OEE a partir de la información recibida desde la planta.

Manufacturing Execution System

React.js · Node.js · Microsoft Azure

Arquitectura del Sistema

El sistema MES fue desarrollado utilizando React.js para la interfaz de usuario y Node.js para los servicios del servidor. Toda la información se almacena en una base de datos Microsoft SQL Server alojada en Microsoft Azure, permitiendo el acceso centralizado desde las diferentes aplicaciones del proyecto.

Frontend

Dashboard interactivo desarrollado en React.js para la visualización de indicadores de producción.

Backend

Servicios desarrollados en Node.js para gestionar la comunicación con la base de datos y el procesamiento de la información.

Base de Datos

Microsoft SQL Server hospedado en Azure como punto central de almacenamiento e intercambio de datos.

Integración con la Planta

Ignition actualiza continuamente la información operativa de la línea de producción dentro del servidor SQL en Azure, registrando variables como el estado de la maquinaria, eficiencia de operación, cantidad actual de botellas producidas y la meta de producción establecida.

El sistema MES consulta esta información en tiempo real para calcular el indicador Overall Equipment Effectiveness (OEE), integrando automáticamente los valores de disponibilidad, desempeño y calidad en un único indicador que resume la eficiencia global del proceso productivo.

Adicionalmente, Ignition puede modificar directamente el indicador de calidad almacenado en la base de datos SQL. Una vez actualizado este valor, el MES recalcula el OEE y actualiza automáticamente los gráficos e indicadores mostrados en el dashboard.
