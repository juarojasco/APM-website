COMUNICACIONES INDUSTRIALES

Arquitectura de Comunicaciones del Sistema

La integración de las diferentes herramientas desarrolladas durante el proyecto se realizó mediante una arquitectura de comunicaciones que permite el intercambio de información entre el PLC, el gemelo digital, la celda robotizada, el sistema SCADA, la plataforma MES y los servicios en la nube. Esta infraestructura garantiza la sincronización de datos, el monitoreo en tiempo real y la interoperabilidad entre los diferentes componentes que conforman la solución de Industria 4.0.

RSLinks Classic, Ignition y Node-RED

Plataforma de comunicaciones industriales

¿Cómo se realizó la comunicación?

La integración del proyecto se desarrolló utilizando RSLinks Classic para establecer la comunicación entre Studio 5000 y el PLC, Ignition como plataforma SCADA para la adquisición y supervisión de datos, y Node-RED como middleware encargado de conectar los diferentes componentes del sistema y facilitar el intercambio de información entre el PLC, el gemelo digital, RobotStudio, Azure y el sistema MES.

RSLinks Classic

Configuración del servidor de comunicaciones y acceso a los dispositivos Rockwell Automation.

Ignition

Supervisión del proceso, adquisición de datos y publicación de información hacia el sistema MES.

Node-RED

Integración entre PLC, simuladores, bases de datos SQL, Azure y aplicaciones web.
